BattleGround Timer Bars for AzCastBar
-------------------------------------
Shows timers for various things in battlegrounds.

Supported Timers
----------------
- Time left on an open battleground queue ready to enter.

- AV: Time until the game starts.
- AV: Timers on graveyards and towers.
- AB: Time until the game starts.
- AB: Timers on bases.
- WSG: Time until the game starts.
- WSG: Flag Respawn time.
- EotS: Time until the game starts.
- SotA: Time until the game starts.
